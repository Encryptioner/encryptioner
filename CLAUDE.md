# Encryptioner — GitHub Profile README

This repo (`encryptioner/encryptioner`) is the special GitHub profile README repository. The `README.md` here renders as the GitHub profile page at https://github.com/Encryptioner.

## Structure

- `README.md` — The profile README (rendered on GitHub profile)
- `assets/` — Images used in the README (header, tech logos)

## Editing Guidelines

- Keep the README visually balanced — badges, images, and text sections should feel cohesive
- Always test badge URLs before committing (shields.io can change formats)
- Images in `assets/` are referenced via relative paths (`./assets/...`)
- The portfolio site is at https://encryptioner.github.io
- Career highlights page is at https://encryptioner.github.io/career-highlights/

## Project Links Convention

When adding project entries, use this format:
```markdown

```

## Branch Strategy

- `master` — main branch, directly deployed as GitHub profile
